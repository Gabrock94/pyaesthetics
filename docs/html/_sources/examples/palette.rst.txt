.. _colorpalette:

Extraction of Color Palette from an Image
===========================================

In pyaesthetics, it is possible to extract the palette of an image using the *colorDetection* module.
The function getColorsW3C can be used not only to extract the percentage of colors accordingly to the W3C specification, but can also be used to generate a palette of n colors from an image.
