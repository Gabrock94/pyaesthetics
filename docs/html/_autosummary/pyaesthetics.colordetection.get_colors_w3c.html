pyaesthetics.colordetection.get_colors_w3c¶

pyaesthetics.colordetection.get_colors_w3c(img, ncolors=16, plot=False, plotncolors=5)¶

This function is used to get a simplified color palette (W3C colors). It can be used with 16 (https://www.html-color-names.com/basic-color-names.php) or 140 colors (https://www.w3schools.com/colors/colors_names.asp).

Parameters:

  • img (numpy.ndarray) – image to analyze in RGB

  • ncolors (int) – number of colors to use (16 or 140)

  • plot (boolean) – whether to plot a color palette image

  • plotncolors (int) – number of colors to use in the palette image

Returns:

percentage distribution of colors according to the W3C basic colors

Return type:

list of shape ncolors x 2, where x[0] is the color name and x[1] the percentage of pixels most similar to that color in the image
