pyaesthetics.visualcomplexity.get_visual_complexity_quadtree¶

pyaesthetics.visualcomplexity.get_visual_complexity_quadtree(image, minStd, minSize)¶

Calculate the visual complexity of an image using quadtree decomposition.

Parameters:

  • image – Input image (grayscale).

  • minStd – Minimum standard deviation for splitting blocks.

  • minSize – Minimum size of blocks.

Returns:

Number of blocks after quadtree decomposition.
